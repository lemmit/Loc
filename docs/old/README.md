# docs/old — the archived design corpus

This directory holds the frozen design record: [`proposals/`](proposals/)
(118 formal proposals — problem → surface → grammar → lowering semantics →
open questions) and [`plans/`](plans/) (71 implementation plans and design
notes), moved here 2026-07-13.

**Statuses and orderings in these files are frozen and superseded.**
The live roadmap — every still-open item from this corpus, divided into
feature tracks and agent-pickable missions — is
[`../new-plan/README.md`](../new-plan/README.md), and
[`../new-plan/coverage.md`](../new-plan/coverage.md) maps every file here
to its disposition (shipped / superseded / historical / carried by
mission M-Tx.y).

Keep using these docs for what they're good at: the design reasoning,
grammar sketches, worked examples, and decision trails behind each
feature. Do **not** resurrect their status tables, tier orderings, or
DEBT/backlog registers — update the mission status in `new-plan/` instead.
Pinned decisions still live in [`../decisions.md`](../decisions.md).
